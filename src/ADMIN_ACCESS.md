# 🔐 管理后台访问指南

## 🎯 两种访问方式

### 方式1：点击浮动按钮（最简单）

在手机应用界面的**右上角**，你会看到一个紫色的**"🔧 管理后台"**按钮。

1. 点击这个按钮
2. 页面会自动跳转到管理后台

### 方式2：修改URL

在浏览器地址栏中添加 `#/admin` 到当前URL后面：

```
https://your-app-url.com#/admin
```

例如：
- 如果你的应用URL是：`https://example.com`
- 管理后台URL是：`https://example.com#/admin`

### 返回商店

在管理后台页面，点击右上角的"返回商店"按钮即可返回手机应用界面。

## 管理后台功能

### 📊 仪表盘
- 查看总产品数、订单数、待处理订单和总营收
- 查看最新添加的产品
- 查看最新订单

### 📦 产品管理
- 查看所有产品列表（支持搜索）
- 编辑产品信息
- 删除产品
- 查看产品详细信息

### ➕ 添加产品
完整的产品添加表单，包括：
- 产品名称
- 价格（显示文本和数值）
- 详细描述
- 供应商/农场
- 产地
- 分类
- 图片URL
- 饮食标签（VG/GF/DF）

### 📋 订单管理
- 查看所有订单
- 查看订单详情（客户信息、购买商品）
- 更新订单状态（待处理、处理中、已发货、已送达、已取消）

---

## ⚡ 快速开始

1. **访问管理后台**
   ```
   在地址栏添加: #/admin
   ```

2. **添加第一个产品**
   - 点击"添加产品"标签
   - 填写产品信息
   - 点击"添加产品"按钮

3. **查看订单**
   - 点击"订单管理"标签
   - 查看用户提交的订单
   - 更新订单状态

## 💡 使用建议

### 桌面优先设计
管理后台专为桌面浏览器设计，建议使用：
- Chrome
- Firefox
- Safari
- Edge

### 响应式布局
虽然主要为桌面设计，但管理后台也支持平板等较大屏幕设备。

### 数据持久化
所有通过管理后台添加的产品和订单都会保存在Supabase数据库中，刷新页面不会丢失数据。

## 🎯 界面对比

| 功能 | 手机应用 | 管理后台 |
|------|---------|---------|
| URL | `/` | `/#/admin` |
| 设计 | 移动端（iPhone界面） | 桌面端（全屏） |
| 用途 | 用户购物 | 管理产品和订单 |
| 功能 | 浏览、购买 | 添加、编辑、管理 |
| 界面 | 手机容器 | 响应式网页 |

## ⚠️ 注意事项

1. **无需额外安装** - 管理后台已内置在应用中
2. **无需登录** - 当前版本无认证系统（生产环境建议添加）
3. **数据共享** - 管理后台和手机应用使用同一个数据库
4. **即时生效** - 在管理后台添加的产品会立即在手机应用中显示
